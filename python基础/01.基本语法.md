### 一、字面量
1. 数字
   - 整数(int)
   - 浮点型数(float)
   - 复数(complex)
   - 布尔(bool)
2. 字符串(String)
3. 列表(List): 有序的可变序列
4. 元组(Tuple): 有序不可变序列
5. 集合(Set): 无序不重复集合
6. 字典(Dictionary): 无序key-value集合

### 二、注释
python中的注释符是`"#"`, `"#"`右边的内容都是注释内容
多行注释是使用`"""`对一段话进行注释

### 三、变量
变量: 用于在程序运行时记录值
`变量名称 = 变量的值`
使用`type(需要查看的变量名)`来查看变量存储的数据类型

### 四、类型转换
1. `int(x)` 将x转换为一个整数
2. `float(x)` 将x转化为一个浮点数(浮点数转化为整数丢失精度)
3. `str(x)` 将对象x转化为字符串

### 五、标识符
只允许
- 中文(通常不用)
- 英文
- 数字
- 下划线
其余不允许，大小写敏感，不可以使用关键字，标识符多个单词用下划线隔开

### 六、算数运算符
- `+` 加
- `-` 减
- `*` 乘
- `/` 除
- `//` 整数除
- `%` 取余
- `**` 指数(`a**b`为a的b次方)

### 七、字符串的定义方法
1. 单引号: `name = 'xxx'`
2. 双引号: `name = "xxx"`
3. 三引号: `name = """xxx"""`
若字符串内容带引号，如果是单引号可以用双引号声明，若是双引号可以用单引号声明，还可以使用`\`用作转义字符，解除引号的作用
`name = "\'xxx\'"`

### 八、字符串拼接
```py
name = "xxx"
print("名字是" + name)
```
Java中自带类型转换，所以可以将字符串和数字拼在一起，python不行

### 九、字符串格式化
通过如下语法，完成字符串拼接:
```python
name = "xxx"
age = 12
# 自动将数字类型转化为字符串类型
message = "名字是 %s，年龄是 %s" % (name, age)
print(message)
``` 
`%`表示要占位，s表示要占位的地方
多个变量拼接可以使用括号将变量括起来
其中:
`%s`表示字符串占位
`%d`表示整数占位
`%f`表示浮点型占位

还有一种方式也可以完成字符串拼接
`f"内容{变量}"`
例如:
```py
name = "xxx"
age = 10
# 数字没有精度控制，原样输出
print(f"名字是{name}, 年龄为{age}")
```

### 十、数字精度控制
使用`m.n`来控制数据的宽度和精度
如`%5.2f`就是将数字设置为宽度为5，小数点为2，宽度的计算包括正数、小数、小数点，不够的用空格补上

```py
print(f"1 * 2的结果是: {1 * 2}")

```

### 十三、if语句
语法:
```py
if 要判断的条件:
   条件成立时要做的事 # python使用缩进来判断语句的归属
print("该语句不受if条件约束")
```
if else语句：
语法：
```py
if 条件:
   条件满足时要做的事情
   ...
else:
   条件不满足时做的事情
```
if elif else语句:
语法：
```py
if 条件1:
   条件1满足时该干的事
elif 条件2:
   条件2满足应当做的事情
else:
   所有条件都不满足应该做的事情
```
判断语句嵌套:
判断语句可以自由组合嵌套，但要注意严格缩进
```py
if 条件1:
   # 条件1满足执行条件2判断
   if 条件2:
      条件2满足时执行的操作
   else:
      条件2不满足时执行操作
else:
   条件1不满足时执行的操作
```

### 十四、循环语句
1. while循环:
语法:
```py
while 条件:
   条件满足时执行的条件
   # 只要条件满足，就会一直执行
```
while循环嵌套:
```py
while 条件1:
   while 条件2:
      执行语句
   执行语句
```
2. for循环
```py
# 若要规范地访问x，可在开头定义i = 0
for x in 待处理数据集:
   循环满足时执行的代码
print(x) # 关于x在规范上不允许在循环外部访问，不过实际上是可与访问到的
```
3. range语句
生成一个简单的数字序列:
```py
range(5) # 该语句取出的数据是[0, 1, 2, 3, 4]
range(5, 10) # 生成一个从num1到num2(不包含num2)的数字序列[5,6,7,8,9]
range(1, 10, 2) # 生成一个从num1到num2(不包含num2)，步数为2的数字序列[1,3,5,7,9]
```
4. continue和break:
   - `continue`是指跳出本次循环，重新开始循环
   - `break`是指直接结束整个循环，跳出循环
