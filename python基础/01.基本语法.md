### 一、字面量
1. 数字
   - 整数(int)
   - 浮点型数(float)
   - 复数(complex)
   - 布尔(bool)
2. 字符串(String)
3. 列表(List): 有序的可变序列
4. 元组(Tuple): 有序不可变序列
5. 集合(Set): 无序不重复集合
6. 字典(Dictionary): 无序key-value集合

### 二、注释
python中的注释符是`"#"`, `"#"`右边的内容都是注释内容
多行注释是使用`"""`对一段话进行注释

### 三、变量
变量: 用于在程序运行时记录值
`变量名称 = 变量的值`
使用`type(需要查看的变量名)`来查看变量存储的数据类型

### 四、类型转换
1. `int(x)` 将x转换为一个整数
2. `float(x)` 将x转化为一个浮点数(浮点数转化为整数丢失精度)
3. `str(x)` 将对象x转化为字符串

### 五、标识符
只允许
- 中文(通常不用)
- 英文
- 数字
- 下划线
其余不允许，大小写敏感，不可以使用关键字，标识符多个单词用下划线隔开

### 六、算数运算符
- `+` 加
- `-` 减
- `*` 乘
- `/` 除
- `//` 整数除
- `%` 取余
- `**` 指数(`a**b`为a的b次方)

### 七、字符串的定义方法
1. 单引号: `name = 'xxx'`
2. 双引号: `name = "xxx"`
3. 三引号: `name = """xxx"""`
若字符串内容带引号，如果是单引号可以用双引号声明，若是双引号可以用单引号声明，还可以使用`\`用作转义字符，解除引号的作用
`name = "\'xxx\'"`

### 八、字符串拼接
```py
name = "xxx"
print("名字是" + name)
```
Java中自带类型转换，所以可以将字符串和数字拼在一起，python不行

### 九、字符串格式化
通过如下语法，完成字符串拼接:
```python
name = "xxx"
age = 12
# 自动将数字类型转化为字符串类型
message = "名字是 %s，年龄是 %s" % (name, age)
print(message)
``` 
`%`表示要占位，s表示要占位的地方
多个变量拼接可以使用括号将变量括起来
其中:
`%s`表示字符串占位
`%d`表示整数占位
`%f`表示浮点型占位

还有一种方式也可以完成字符串拼接
`f"内容{变量}"`
例如:
```py
name = "xxx"
age = 10
# 数字没有精度控制，原样输出
print(f"名字是{name}, 年龄为{age}")
```

### 十、数字精度控制
使用`m.n`来控制数据的宽度和精度
如`%5.2f`就是将数字设置为宽度为5，小数点为2，宽度的计算包括正数、小数、小数点，不够的用空格补上

### 十一、表达式格式化
例如:
```py
print("1 * 2的结果: " % (1 * 2))
print(f"1 * 2的结果是: {1 * 2}")
print("字符串在Python的类型是: %s" % type("字符"))
```
