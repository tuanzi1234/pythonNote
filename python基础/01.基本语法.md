### 一、字面量
1. 数字
   - 整数(int)
   - 浮点型数(float)
   - 复数(complex)
   - 布尔(bool)
2. 字符串(String)
3. 列表(List): 有序的可变序列
4. 元组(Tuple): 有序不可变序列
5. 集合(Set): 无序不重复集合
6. 字典(Dictionary): 无序key-value集合

### 二、注释
python中的注释符是`"#"`, `"#"`右边的内容都是注释内容
多行注释是使用`"""`对一段话进行注释

### 三、变量
变量: 用于在程序运行时记录值
`变量名称 = 变量的值`
使用`type(需要查看的变量名)`来查看变量存储的数据类型

### 四、类型转换
1. `int(x)` 将x转换为一个整数
2. `float(x)` 将x转化为一个浮点数(浮点数转化为整数丢失精度)
3. `str(x)` 将对象x转化为字符串

### 五、标识符

- 数字
- 下划线

### 六、算数运算符
- `+` 加
- `-` 减
- `*` 乘
- `/` 除
- `//` 整数除
- `%` 取余
- `**` 指数(`a**b`为a的b次方)

### 七、字符串的定义方法
1. 单引号: `name = 'xxx'`
2. 双引号: `name = "xxx"`
3. 三引号: `name = """xxx"""`
若字符串内容带引号，如果是单引号可以用双引号声明，若是双引号可以用单引号声明，还可以使用`\`用作转义字符，解除引号的作用
`name = "\'xxx\'"`

### 八、字符串拼接
```py
name = "xxx"
print("名字是" + name)
```
Java中自带类型转换，所以可以将字符串和数字拼在一起，python不行

### 九、字符串格式化
通过如下语法，完成字符串拼接:
```python
name = "xxx"
age = 12
# 自动将数字类型转化为字符串类型
message = "名字是 %s，年龄是 %s" % (name, age)
print(message)
``` 
`%`表示要占位，s表示要占位的地方
多个变量拼接可以使用括号将变量括起来
其中:
`%s`表示字符串占位
`%d`表示整数占位
`%f`表示浮点型占位

还有一种方式也可以完成字符串拼接
`f"内容{变量}"`
例如:
```py
name = "xxx"
age = 10
# 数字没有精度控制，原样输出
print(f"名字是{name}, 年龄为{age}")
```

### 十、数字精度控制

### 十三、if语句
语法:
```py
if 要判断的条件:
   条件成立时要做的事 # python使用缩进来判断语句的归属
print("该语句不受if条件约束")
```
if else语句：
语法：
```py
if 条件:
   条件满足时要做的事情
   ...
else:
   条件不满足时做的事情
```
if elif else语句:
语法：
```py
if 条件1:
   条件1满足时该干的事
elif 条件2:
   条件2满足应当做的事情
else:
   所有条件都不满足应该做的事情
```
判断语句嵌套:
判断语句可以自由组合嵌套，但要注意严格缩进
```py
if 条件1:
   # 条件1满足执行条件2判断
   if 条件2:
      条件2满足时执行的操作
   else:
      条件2不满足时执行操作
else:
   条件1不满足时执行的操作
```

### 十四、循环语句
1. while循环:
语法:
```py
while 条件:
   条件满足时执行的条件
   # 只要条件满足，就会一直执行
```
while循环嵌套:
```py
while 条件1:
   while 条件2:
      执行语句
   执行语句
```
2. for循环
```py
# 若要规范地访问x，可在开头定义i = 0
for x in 待处理数据集:
   循环满足时执行的代码
print(x) # 关于x在规范上不允许在循环外部访问，不过实际上是可与访问到的
```
3. range语句
生成一个简单的数字序列:
```py
range(5) # 该语句取出的数据是[0, 1, 2, 3, 4]
range(5, 10) # 生成一个从num1到num2(不包含num2)的数字序列[5,6,7,8,9]
range(1, 10, 2) # 生成一个从num1到num2(不包含num2)，步数为2的数字序列[1,3,5,7,9]
```
4. continue和break:
   - `continue`是指跳出本次循环，重新开始循环
   - `break`是指直接结束整个循环，跳出循环

### 十五、函数
封装好的，可以直接重复使用的代码段 
1. 函数的定义
```py
def 函数名(参数):
   """
   函数说明文档:
   :param xxx: 对形参的说明
   :return: 返回值的说明
   """
   函数体
   return 返回值
```
调用: `函数名(参数)`
若要传递多个参数使用逗号隔开
当不特别声明函数返回值时，返回值的类型为None
2. 嵌套调用
函数里又调用了另外一个函数
```py
def a():
   函数体a
def b():
   函数体b
   a()
# 调用b
b()
```
3. 函数作用域:
```py
# 局部变量
def a():
   num1 = 100
   print(num)
a()
print(num1) # 报错，无法在函数外使用该变量

print("------------------------")

num2 = 200
def b():
   # 若要在其中使用num2 = 300, 则只会在b()作用域中生效
   # 若要使其在全局修改，则使用global num2 声明num2为全局变量
   print(num2)
b()
print(num2) # 不报错，全局变量可以随意使用
```

### 十七、数据容器 
一种可以容纳多个数据的数据类型，其中的每一份数据称之为一个元素
元素可以是任意类型变量
1. 列表
- 下标索引: 每个元素都有下表索引，从左往右，从0开始递增；从右往左，从-1开始递减
- 列表特点：
  - 容纳多个元素(上限为2^63 - 1个)
  - 容纳不同类型的元素
  - 数据有序存储
  - 允许重复数据存在
  - 可以修改
```py
# 1. 列表的定义
# 1.1 字面量
[元素1, 元素2, 元素3, ...]
# 1.2 定义变量
变量名 = [元素1, 元素2, 元素3]
# 1.3 定义空列表
变量名 = []
变量名 = list()
```

- 列表索引
```py
# 2. 列表索引
# 2.1 利用下标索引取出元素(从左向右)
print(变量名[0]) # 取出元素1
print(变量名[1]) # 取出元素2

# 2.2 利用下标索引取出元素(从右向左)
print(变量名[-1]) # 取出元素3
print(变量名[-2]) # 取出元素2

# 2.3嵌套列表的情况
list1 = [[1 , 2] , [3 , 4]]
print(list1[0][0]) # 取出1
```

- 关于list的方法
```py
# 3. 关于list的方法
# 3.1 查找元素在列表的下标
test_list = [1,2,3,4,5,6]
print(f"1的下标为{test_list(1)}")

# 3.2 修改列表的值
test_list = [1,2,3,4,5,6]
test_list[0] = 10
print(test_list) # [10,2,3,4,5,6]

# 3.3 在列表中插入元素
test_list = [1,2,3,4,5,6]
test_list.insert(1, 9) 
print(test_list) # [1,9,2,3,4,5,6]

# 3.4 追加元素(将元素插入列表尾部)
test_list = [1,2,3,4,5,6]
test_list.append(9) 
print(test_list) # [1,2,3,4,5,6,9]

# 3.5 追加一批元素
test_list = [1,2,3,4,5,6]
test.extend([13,14,15])
print(test_list) # [1,2,3,4,5,6,13,14,15]

# 3.6 删除元素
test_list = [1,2,3,4,5,6]
del test_list[0]
x = test_list.pop(1)
print(test_list) # [2,4,5,6]
print(f"取出的元素是{x}") # 输出为3, pop可以将元素作为返回值得到

# 3.7 从前到后搜索元素，删除第一个匹配项
test_list = [1,2,1,2,1,2,1,2]
test_list.remove(2)
print(test_list) # [1,1,2,1,2,1,2]

# 3.8 清空整个列表
test_list.clear()
print(test_list) # []

# 3.9 统计表内元素的数量
test_list = [1,2,1,2,1,2,1,2]
test_list.count(2)

# 3.10 统计列表中实际的元素数量
print(len(test_list)) # 8
```
